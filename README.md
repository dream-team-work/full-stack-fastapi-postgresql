## PARA RODAR O TESTE

na raiz do projeto com o docker instalado em sua maquina rode o comando:
````
docker-compose up 
````

## ACESSE O LINK DA APLICACAO COM A URL ABAIXO:
````
http://localhost:8000/docs#/
````
